# EduGamificación: Alfabetización para Docentes

Este es un espacio de formación diseñado para profesores que desean integrar dinámicas de juego en sus procesos de enseñanza. La plataforma ofrece una ruta de aprendizaje estructurada en 5 secciones clave con autoevaluaciones integradas.

## 🚀 Objetivo del Proyecto
Cerrar la brecha digital y pedagógica mediante el uso de la gamificación, permitiendo que los docentes creen experiencias de aprendizaje más atractivas y motivadoras para sus estudiantes.

## 📚 Estructura de Contenidos
1. **Introducción a la Gamificación:** Conceptos básicos y beneficios.
2. **Mecánicas y Dinámicas:** Puntos, niveles, insignias y tablas de clasificación.
3. **Herramientas Digitales:** Uso de plataformas para gamificar el aula.
4. **Diseño de Experiencias:** Cómo planificar una unidad didáctica gamificada.
5. **Evaluación y Feedback:** Medir el impacto y progreso del estudiante.

## 🛠️ Configuración del Entorno
Este repositorio está vinculado con **Qwen Coder** para la generación y despliegue de código.
